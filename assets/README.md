Save your files in these locations:

## App icon & logo
| File | Path |
|------|------|
| App icon / logo | `assets/images/logo.png` |
| Favicon | `assets/images/favicon.ico` |
| Social preview (Facebook, etc.) | `assets/images/og-image.png` |

## Screenshots
| File | Path |
|------|------|
| Hero phone screenshot | `assets/images/screenshot-hero.png` |
| Gallery screenshot 1–4 | `assets/images/screenshot-1.png` … `screenshot-4.png` |

**Recommended size:** 360×720 px (9:16 phone ratio), PNG or JPG.

## APK download
| File | Path |
|------|------|
| Android APK | `assets/downloads/maya.apk` |

After adding the APK, the download button on the website will work automatically.

## Quick checklist
- [ ] `logo.png` — square app icon (512×512 recommended)
- [ ] `favicon.ico` — browser tab icon
- [ ] `screenshot-hero.png` — main app screen for hero section
- [ ] `screenshot-1.png` … `screenshot-4.png` — gallery images
- [ ] `maya.apk` — installable Android file
- [ ] `og-image.png` — optional, for link previews (1200×630 px)
